Proyecto Calculadora para numeros naturales
-------------------------------------------
-------------------------------------------
Pila de Pruducto
----------------
Historia 1.  Poder sumar dos números naturales para
poder conocer el valor resultado.

Historia 2.Poder multiplicar dos números naturales
para poder conocer el valor resultado.

Historia 3.Poder restar dos números naturales para
poder conocer el valor resultado. El resultado debe ser otro
número natural.

Historia 4.Como usuario quiero saber cuándo no se puede realizar una
resta para poder saber que me he equivocado.

Historia5.Poder utilizar notación de
números romanos para poder utilizarlo como herramienta de
apoyo para los niños.


Sprint 1 :								Estimación
--------------------------------------------------
Historia 1:	H_Suma									
  Tarea 1: Escribir las pruebas             | 1
  Tarea 2: Ecribir el código				| 1
  Tarea 3: Hacer las pruebas y refactorizar | 2
  
Historia 2: H_Multiplicacion
  Tarea 1: Escribir las pruebas             | 1
  Tarea 2: Ecribir el código				| 1
  Tarea 3: Hacer las pruebas y refactorizar | 2
Historia 3: H_Resta
  Tarea 1: Escribir las pruebas             | 1
  Tarea 2: Ecribir el código				| 1
  Tarea 3: Hacer las pruebas y refactorizar | 2
Historia 4: H_Resta_imposible
  Tarea 1: Escribir las pruebas             | 1
  Tarea 2: Ecribir el código				| 1
  Tarea 3: Hacer las pruebas y refactorizar | 2

  
  Notas:
  --------
Historia 3 es bloqueante para historia 4. 
Cada tarea de escribir pruebas se puede hacer en paralelo con la tarea de escribir codigo, porque interesa 
que se hagan por diferentes personas. Una vez acabadas estas dos, se puede proceder a la tarea 3 correspondiente de hacer las pruebas y refactorizar.
